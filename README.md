# Emergency Response App

A modern web application for reporting and managing emergency incidents in real-time.

## Features (add yours here)
- User authentication & role-based access
- Real-time incident reporting (location, description, media)
- Admin dashboard for authorities
- Database-powered with Prisma ORM
- Responsive UI with Tailwind CSS

## Tech Stack
- **Frontend/Fullstack**: Next.js (App Router) + TypeScript + React
- **Styling**: Tailwind CSS
- **Database/ORM**: Prisma
- **Deployment**: Vercel
